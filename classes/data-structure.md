# Data Structure

The **Data Structure** course introduces fundamental data structures and
algorithms that support efficient computation, with a focus on  
**implementation**, **complexity analysis**, and **practical usage**.

## Main Topics
- Arrays, linked lists, stacks, and queues  
- Trees, binary search trees, and heaps  
- Hash tables and dictionaries  
- Basic graph structures and traversal algorithms  

## Course Features
- Coding assignments in C/C++ or Python  
- Analysis of time and space complexity  
- Examples connected to real systems and CPS

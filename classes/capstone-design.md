# Capstone Design

The Capstone Design course provides students with an opportunity to apply their knowledge of **software engineering, AI, and cyber-physical systems** to real-world problems.  
Students work in teams and develop a complete system from problem definition to final demonstration.

## Course Overview
- Problem definition and proposal writing  
- Requirement analysis and system specification  
- Software architecture and system design  
- Implementation, integration, and testing  
- Mid-term review, final presentation, and demo day  

## Project Examples
- Medical AI decision-support tools (e.g., ophthalmology assistants)  
- Autonomous robot and F1TENTH-based applications  
- Web dashboards for data visualization and monitoring  
- Real-time sensing and safety monitoring systems
